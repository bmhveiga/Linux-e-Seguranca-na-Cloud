# WordPress

Esta pasta documenta a publicação WordPress realizada no Nível 3 - Avançado.

A instalação real do WordPress foi publicada no servidor Linux em:

/var/www/html/topico-03-wordpress

Os ficheiros completos do WordPress não foram adicionados ao repositório, porque pertencem à aplicação WordPress e podem incluir configurações locais. O repositório contém a documentação, comandos utilizados, validação e evidências da publicação.
